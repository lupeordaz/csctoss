Diff - unix/linux utility  


diff Command:

    diff analyzes two files and prints the lines that are different. Essentially, it outputs a set 
    of instructions for how to change one file to make it identical to the second file.

Using diff to extract 

    diff -u mac1 mac2 | grep '^-[^-]' | sed 's/^-//'

Display file differences side by side

    From man diff, you can use -y to do side-by-side.

      -y, --side-by-side
             output in two columns

    [gordaz@cctlix03 5653]$ diff -y mac2 mac1 
    3490                3490
                        > 3824
                        > 4106
    4642                4642
                        > 4769
                        > 4775
                        > 4844
                        > 4871
    4909                4909
                        > 5231

Display file entries from file1 that don't appear in file2

    diff -u mac1 mac2 | grep '^-[^-]' | sed 's/^-//'
