Display IP Address

  On Centos:
    ip addr 
    1: lo: %ltLOOPBACK,UP,LOWER_UP%gt mtu 65536 qdisc noqueue state UNKNOWN
        link/loopback 00:00:00:00:00:00 brd 00:00:00:00:00:00
        inet 127.0.0.1/8 scope host lo
        inet6 ::1/128 scope host
           valid_lft forever preferred_lft forever
    2: eth0: %ltBROADCAST,MULTICAST,UP,LOWER_UP%gt mtu 1500 qdisc pfifo_fast state UP qlen 1000
        link/ether 00:0c:29:b3:50:e1 brd ff:ff:ff:ff:ff:ff
        inet 10.17.72.21/29 brd 10.17.72.23 scope global eth0
        inet6 fe80::20c:29ff:feb3:50e1/64 scope link
           valid_lft forever preferred_lft forever

      The bold ip address is the external ip address.

    hostname -I
    10.17.72.21


  To get the external IP address, use the following curl command to call checkip.dyndns.org 
    and display IP address from outside source:
    
    curl -s checkip.dyndns.org | sed -e 's/.*Current IP Address: //' -e 's/<.*$//'
