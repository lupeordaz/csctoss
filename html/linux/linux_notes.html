Kill a user session  

  Must be su to execute this command.

  pkill -o -u <user name> sshd

  To kill the session of user lperrete, enter the following:
  
  pkill -o -u lperrete sshd

List all the listening ports

    netstat -a | grep LIST

    For apache listeners:

    netstat -A inet -lnp

Display exernal IP address
    
    Use the following curl command to call checkip.dyndns.org and display IP address:
    
    curl -s checkip.dyndns.org | sed -e 's/.*Current IP Address: //' -e 's/<.*$//'
