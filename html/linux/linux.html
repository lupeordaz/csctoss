  Retype new password: 

  [root@gemisys ~]# passwd -e lordaz
  Expiring password for user lordaz.
  passwd: Success

Determine Physical Memeory
  dmidecode -t 17 | grep "Size.*MB" 
    Result is 
      Size: 8192 MB
      Size: 4096 MB
  Or
  dmidecode -t 17 | grep "Size.*MB" | awk '{s+=$2} END {print s / 1024}'
    Result is 
    12
  Or
  cat /proc/meminfo | grep MemTotal | awk '{ print $2 }'
    Result is 
    12199260
