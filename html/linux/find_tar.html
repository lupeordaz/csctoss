Unix/Linux Find Utility  

find Commands: 



  

    Purpose
    Command
  

  

    Search file by name
    find . -name filename.txt
  

  

    Search file by type/extension
    find . -name "*.txt"
  

  

    Search case insensitive
    find -iname testfile1.txt
  

  

    Search files that don't match search pattern
    find . -not -name "*.txt"
  

  

    Search for file but ignore "Permission denied" message
    find . -name art  2>&1 | grep -v "Permission denied"
  

  

    Recursively look for files with a specific extension
    find <$directory> -type f -name "*.in"
