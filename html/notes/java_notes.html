Notes on Java and Java utilities  


    
    Display current version of java:
        java -version
        
    Display contents of a jar file:    
        jar tf <jar file name>

    Find a specific class in a jar
        jar tf <jar file name> | grep <class name>
    
    Find the class GroovyStarter in a jar library
        First cd to the directory,i.e. cd /usr/local/groovy/lib
        find . -name '*.jar' -print |  xargs  -I '{}' sh -c '/usr/java6/bin/jar tf {} | grep <GroovyStarter> &&  echo {}'

        Result:
        org/codehaus/groovy/tools/GroovyStarter.class
        ./groovy-2.3.4.jar

        Displays result that the jar groovy-2.3.4.jar contains the class
