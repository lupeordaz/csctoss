  Using the files imported from SourcForge, the server comes with two user id's, root and ubuntu

Setting up the shared folders
     
    
  Log in as root to do the following.

  1.  mkdir /media/cdrom
  2.  mount -r /dev/cdrom /media/cdrom
  3.  yum update
  4.  yum insstall gcc
  5.  yum install kernel-devel
  6.  yum update kernel*
  7.  rpm -Uvh http://dl.fedoraproject.org/pub/epel/7/x86_64/e/epel-release-7-5.noarch.rpm
  8.  rpm -Uvh http://dl.fedoraproject.org/pub/epel/6/i386/epel-release-6-8.noarch.rpm
  9.  rpm -Uvh http://dl.fedoraproject.org/pub/epel/5/i386/epel-release-5-4.noarch.rpm
 10.  yum install gcc kernel-devel kernel-headers dkms make bzip2 perl
 11.  add KERN_DIR environment variable: KERN_DIR=/usr/src/kernels/`uname -r`
