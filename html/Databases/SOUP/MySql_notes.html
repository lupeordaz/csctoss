    Install MySql via yum
  
      
    
    1. cd /tmp -- retrieve to this directory
      2. 
    
    2. wget http://repo.mysql.com/mysql-community-release-el7-5.noarch.rpm
      3. 
    
    3. rpm -ivh mysql-community-release-el7-5.noarch.rpm
      4. 
    
    4. ls /etc/yum.repos.d/mysql* -- should return two files:
          /etc/yum.repos.d/mysql-community.repo  
          /etc/yum.repos.d/mysql-community-source.repo 
      5. 
    
    5. yum install mysql-server -- Answer 'y' to all prompts
      6. 
  
    
  Should return a "Complete!" message.

How to start/stop/restart MySql Server
  To Start:   systemctl start mysqld

  To Stop:    systemctl stop mysqld

  To Restart: systemctl restart mysqld

  GRANT ALL PRIVILEGES ON drupal.* TO ’tca’@‘localhost';

Start and Stop mysql
  Stop MySql

    sudo /etc/init.d/mysqld stop

  Start in safe mode bypassing password

    sudo mysqld_safe --skip-grant-tables & 

Connect as root
    mysql -u root

    use mysql;

Recovering Password
    update user set password=PASSWORD("mynewpassword") where User='root';

    flush privileges;

    quit

    mysql stop

    mysql start

Logon as root and prompt for password
    mysql -u root -p
