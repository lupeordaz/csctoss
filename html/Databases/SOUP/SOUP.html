SOUP Terminology  


    
Systech Online Update Protocol

SDM
  Systech Device Monitor provides three basic functions using two protocols:

    - Remote Access (rHTTP)
    - Automatic Updates (SOUP - Systech Online Update Protocol)
    - Unit Monitoring (SOUP heratbeats)

PRL
  Preferred Roaming List is a way of telling your device (phone, router,etc.)
  to look for a signal so that you get faster, better connections.

MySql login string

  mysql -u root sdm

  This logs in to the sdm database in MySql on SOUP server.  Be careful.

SOUP admin scripts from Systech

  Scripts are located in /opt/soup-adminscripts/

  Logs are in /opt/sdm
