Have 5 devices with corrupt identifiers.  We need to know what the old values were before they were changed.


csctoss=# SELECT * 
            FROM change_log 
           WHERE table_name = '"csctoss"."unique_identifier"' 
             AND primary_key LIKE '41313,M%' 
             AND column_name <> 'MAC ADDRESS' 
           ORDER BY change_timestamp;
